A step-by-step guide to creating, customizing, and running a basic "Hello World" application in the latest **Android Studio Narwhal**.

üåü What is 'Hello World'?

The "Hello World" program is a classic first step in learning any new programming language or platform. For Android, it's a simple application that displays the text "Hello World!" on the screen of an Android device or emulator. This guide will walk you through setting up a new project in **Android Studio Narwhal** and running this foundational app.

Ensure you have Android Studio Narwhal installed and an Android Emulator set up before starting this guide.

üõ†Ô∏è Creating Your 'Hello World' Project

Let's begin by creating a new Android Studio project for our "Hello World" app using **Android Studio Narwhal**.

  1. Start a New Project:
    • Open **Android Studio Narwhal**.
    • From the **Welcome screen**: Click New Project.
    • From an **open project**: Go to File > New > New Project....
  2. Choose a Project Template:

    In the "New Project" wizard, select the Empty Activity template for Phone and Tablet. This template provides a minimal foundation with a single screen for your app. Click Next.

  3. Configure Your Project:

    On the next screen, provide the following details:

    • Name: Enter MyHelloWorldApp (or any other name you prefer).
    • Package name: This will be automatically generated based on your name (e.g., com.example.myhelloworldapp).
    • Save location: Choose a directory on your computer where you want to save your project files.
    • Language: Select Kotlin (the modern, recommended language for Android development) or Java.
    • Minimum SDK: Choose a reasonable minimum API level (e.g., API 21 for Android 5.0 Lollipop) to ensure your app runs on a wide range of devices.

    Click Finish. **Android Studio Narwhal** will now set up your project, download necessary dependencies, and build your initial project, which may take a few moments depending on your internet speed.

‚úèÔ∏è Exploring and Customizing the Layout

Your app's user interface is defined in XML layout files. For our "Hello World" app, the primary layout is `activity_main.xml`.

  1. Navigate to the Layout File:

    Once the project loads in **Android Studio Narwhal**, expand the project hierarchy in the "Project" window (usually on the left side) and navigate to: app > res > layout > activity_main.xml.

  2. Understand the Default Layout:

    When you open activity_main.xml, **Android Studio Narwhal** will open it in the Design editor. You'll likely see a TextView element centrally placed with the text "Hello World!". This is the default UI provided by the Empty Activity template.

    You can switch between **Code**, **Split** (Code and Design), and **Design** views using the tabs at the top right of the editor window. In the `Code` view, you'll see the XML that defines this layout. It should look something like this:

    <?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="24sp"
        android:textColor="@android:color/black"
        app:layout_constraintBottom_toBottomOf="parent"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintTop_toTopOf="parent" />

</androidx.constraintlayout.widget.ConstraintLayout>

    We've added android:textSize="24sp" and android:textColor="@android:color/black" to make the text more prominent.

  3. (Optional) Change the Text:

    If you want to change the text, you can directly edit the android:text="Hello World!" attribute in the XML to something else, for example: android:text="Greetings from Gemini!".

‚ñ∂Ô∏è Running Your App on an Emulator

Now that your app's layout is ready, let's run it on the Android emulator you've previously set up using **Android Studio Narwhal**.

  1. Ensure Your Emulator is Running:

    Make sure your Android emulator (AVD) is already launched and running. If not, open the **Device Manager** (AVD Manager) in **Android Studio Narwhal** and click the ‚ñ∂ (Play) button next to your desired AVD.

  2. Select the Target Device:

    In the **Android Studio Narwhal** toolbar, there's a dropdown menu showing available devices. Your running emulator should appear here. Select it as your target device.

  3. Run the Application:

    Click the **‚ñ∂ (Run 'app')** button (a green triangle icon) in the **Android Studio Narwhal** toolbar. Android Studio will now compile your application, install it on the selected emulator, and launch it.

You should now see your "MyHelloWorldApp" running on the emulator, displaying "Hello World!" (or your custom message) on the screen. Congratulations, you've successfully created and run your first Android application using **Android Studio Narwhal**! üéâ
