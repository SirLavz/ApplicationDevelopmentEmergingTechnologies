JDK Installation Guide

Week 3 - Downloading and installing the Java Development Kit (JDK) on Windows, macOS, and Linux.

ðŸ“¥ Downloading the JDK

  1. Visit the official Oracle Java website or OpenJDK website.
  2. Choose the appropriate installer for your operating system:
    • Windows: Look for a .exe or .msi installer file.
    • macOS: Select a .dmg or .pkg file.
    • Linux: Download a .tar.gz (compressed archive), .deb (for Debian/Ubuntu), or .rpm (for Fedora/CentOS) file.
  3. Accept the license agreement if prompted and download the file.

ðŸ’» Installation Steps by Operating System

Windows

  1. Locate the downloaded .exe or .msi file and double-click to run the installer.
  2. Follow the prompts in the installation wizard. It's generally safe to accept the default installation path, which is typically C:\Program Files\Java\jdk-<version>.
  3. Once the installation is complete, you will need to set up environment variables.
    • Search for "Environment Variables" in the Windows search bar and open the "Edit the system environment variables" window.
    • Click the "Environment Variables..." button.
    • Under "System variables", find the Path variable and click "Edit".
    • Click "New" and add the path to your JDK's bin directory (e.g., C:\Program Files\Java\jdk-<version>\bin).
    • Click "OK" on all windows to save the changes.
    • Optionally, you can also create a new JAVA_HOME variable with the value set to the JDK installation directory (e.g., C:\Program Files\Java\jdk-<version>).

macOS

  1. Locate the downloaded .dmg file and double-click it.
  2. Drag the JDK icon into the Applications folder, or if it's a .pkg installer, double-click it and follow the on-screen instructions. The JDK is typically installed in /Library/Java/JavaVirtualMachines/.
  3. Open Terminal and set the JAVA_HOME environment variable by editing your shell's configuration file (.zshrc for Zsh or .bash_profile for Bash).
    • Type nano ~/.zshrc (or nano ~/.bash_profile) to open the file in a text editor.
    • Add the following lines to the end of the file: 
      export JAVA_HOME=$(/usr/libexec/java_home)
export PATH="$JAVA_HOME/bin:$PATH"
    • Save and exit the editor, then reload the configuration by running source ~/.zshrc (or source ~/.bash_profile).

Linux

The easiest way is to use your distribution's package manager:

  • Ubuntu/Debian: sudo apt update && sudo apt install default-jdk
  • Fedora/CentOS: sudo dnf install java-latest-openjdk-devel

After installation, you may need to set the JAVA_HOME and PATH environment variables:

  1. Open your shell's configuration file (~/.bashrc or ~/.zshrc) and add these lines: 
    export JAVA_HOME=$(readlink -f /usr/bin/java | sed "s:bin/java::")
export PATH=$JAVA_HOME/bin:$PATH
  2. Save, exit, and reload the configuration with source ~/.bashrc (or source ~/.zshrc).

âœ… Verifying the Installation

After completing the installation, open a new terminal or Command Prompt and run the following commands to ensure that the JDK is installed and configured correctly:

  • java -version
  • javac -version

Both commands should display the installed Java version number without any errors. If they do, your JDK setup is complete! Congratulations! ðŸŽ‰
